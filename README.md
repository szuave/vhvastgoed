# VH Vastgoed

Officiële website voor **Van Hulle Vastgoed** — erkend vastgoedmakelaar in het Meetjesland.

## Over

VH Vastgoed is een vastgoedkantoor opgericht door Nico Van Hulle, gevestigd in Lembeke. De website biedt:

- Overzicht van vastgoed te koop en te huur
- Diensten: verkopen, verhuren, aankoopbegeleiding, gratis schatting
- Beheerdashboard voor de makelaar (panden, foto's, documenten, team)

## Tech

- HTML / CSS / Vanilla JavaScript
- Supabase (database, authenticatie, storage)
- Google Fonts (Montserrat + Cormorant Garamond)
- Font Awesome 6.5
